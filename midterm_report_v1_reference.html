01 動機

養殖廢水中的抗生素危機

全球背景

  • 中國是全球最大抗生素生產與消費國，也擁有最大的海水養殖業
  • 四環素類（TCs）被大量用於水產養殖中的疾病防治與促進生長
  • 缺乏抗生素排放標準 → 養殖廢水直接排入環境

四環素（Tetracycline, TC）的危害

  • 促進抗藥性基因（ARGs）擴散
  • 通過食物鏈傳遞，威脅人類健康
  • 破壞水生生態系統微生物群落
  • 傳統廢水處理技術難以有效去除，可能產生更毒副產物

台灣的相關性

  • 台灣水產養殖業發達（石斑、虱目魚、蝦類等）
  • 養殖用藥管理是重要的食安與環保議題
  • 本研究直接使用天然海水作為溶劑，貼近真實養殖環境

光催化技術的優勢與挑戰

抗生素去除方法比較

方法優點缺點
生物處理成本低對抗生素降解效率低
物理吸附操作簡單不能真正分解，需後續處理
高級氧化（AOP）效率高需化學試劑，成本高
光催化高效、無二次污染、可利用太陽光催化劑回收困難

為什麼選 ZnO？

  • ZnO 是一種n 型半導體，化學穩定性好、成本低
  • 光催化活性高，抗菌性能優異
  • 但能隙 3.12 eV → 僅能被紫外光（< 400 nm）激發
  • 紫外光僅佔太陽光的 ~5% → 可見光利用率極低
  • 光生電子-空穴對複合率高，降低催化效率

改性策略文獻

  • 金屬/金屬氧化物摻雜形成異質結（如 Ag-ZnO, Fe₂O₃/ZnO）
  • 非金屬材料複合（如 g-C₃N₄/ZnO）
  • 先前研究：ZnO 基複合光觸媒去除 TC 最高達 80.18%

01 文獻

為什麼選擇 CuO 摻雜？

CuO 的特性

  • CuO 是p 型半導體（因氧空位缺陷）
  • 能隙僅 1.2 eV → 對可見光有良好響應
  • 成本低廉、製備簡單、化學穩定性優異
  • 具有光化學穩定性，適合長期使用

PN 異質結概念

  • CuO（p 型）+ ZnO（n 型）→ 形成 PN 異質結
  • 內建電場促進光生電子-空穴對的有效分離
  • 降低複合率 → 更多自由基產生 → 更高降解效率

本研究的文獻缺口

  • 大多數研究使用淡水作為溶劑 → 不適用於海水養殖場景
  • 粉體催化劑回收效率低
  • 本研究創新：
    • 使用天然過濾海水作為溶劑
    • 將催化劑負載於聚丙烯多面球（PMSS）上以利回收

01 文獻

粉體催化劑的回收挑戰

粉體光觸媒的問題

  • 奈米粉體催化劑分散在溶液中後，難以完全回收
  • 離心或過濾耗時耗能，不適合大規模應用
  • 殘留的催化劑粉末可能造成二次污染

本研究的解決方案：PMSS 固定化

  • 聚丙烯多面球（PMSS）
  • 將 ZC 光觸媒粉末透過浸漬法固定在 PMSS 表面
  • 球型結構提供大表面積，聚丙烯耐酸鹼、耐海水腐蝕
  • 反應後撈出即可回收，可反覆使用

03 設計

奈米材料設計概念 — PN 異質結

設計思路

將 p 型半導體 CuO 與 n 型半導體 ZnO 結合，利用 PN 異質結的內建電場加速光生載子分離。

ZnO（n 型半導體）

  • 纖鋅礦（wurtzite）結構，能隙 3.12 eV
  • 電子為多數載子，提供電子供體角色

CuO（p 型半導體）

  • 單斜晶系結構，能隙 1.2–1.51 eV
  • 空穴為多數載子，因氧空位缺陷形成 p 型特性

PN 異質結優勢

  • 界面內建電場驅動 e⁻ 和 h⁺ 向相反方向移動
  • 有效抑制光生電子-空穴對的複合
  • 更多載子參與光催化反應 → 更多自由基產生
  • 能隙降至 2.78 eV → 可見光全覆蓋（至 900 nm）

05 鑑定

XRD 結晶結構與 XPS 化學態

XRD 分析

  • ZnO 特徵峰：對應 JCPDS 01-070-2551（六方晶系結構）
  • CuO 特徵峰：對應 JCPDS 01-080-1268（單斜晶系），隨摻雜比例增加而逐漸明顯
  • 無其他雜峰 → 樣品純度高
  • 確認 ZnO + CuO 兩相共存的複合結構

XPS 分析

  • Zn 2p：1021.22 eV (2p₃/₂) + 1044.31 eV (2p₁/₂)，確認 Zn²⁺
  • Cu 2p：六個去卷積峰（932.89–962.38 eV），確認 Cu²⁺
  • O 1s：
    • Zn-O 鍵 531.52 eV → ZC₀.₃ 中移至 531.82 eV（正移 0.3 eV）
    • Cu-O 鍵 529.09 eV（新出現）
    • 結合能位移 → ZnO 和 CuO 間存在電子交互作用

O 1s 位移暗示 CuO 引入產生額外氧缺陷，有利光催化反應。

TC 降解途徑與毒性評估

LC-MS/MS 中間產物鑑定

  • 使用三重四極桿液相層析串聯質譜儀分析
  • 鑑定出 9 個中間產物（P1–P9），建立完整降解路徑圖

主要降解路徑

  • 路徑 1：脫氨基、脫甲基反應（P0→P1→P2）
  • 路徑 2：開環反應（芳環斷裂，P3→P4→P5）
  • 路徑 3：小分子化（P6→P7→P8→P9）
  • 最終產物：CO₂ + H₂O（完全礦化）

毒性評估

  • ECOSAR 軟體（美國 EPA 驗證）分析中間產物毒性
  • 大部分中間產物毒性低於原始 TC
  • 最終產物 CO₂ + H₂O 無毒
  • 光催化過程不產生更毒副產物

10 機制

ZnO/CuO 光催化降解機制

PN 異質結光催化過程

  1. ZnO + CuO 形成 PN 異質結，調控能帶結構
  2. 可見光照射 → 電子從 VB 躍遷至 CB → 產生 h⁺
  3. CB 上的 e⁻ 與溶解氧反應 → 生成 ·O₂⁻
  4. ·O₂⁻ 與水反應 → 生成 ·OH
  5. VB 上的 h⁺ 氧化 OH⁻ → 額外生成 ·OH
  6. h⁺ + ·OH + ·O₂⁻ → TC 礦化為 CO₂ + H₂O

反應方程式

(3) ZC + hν → ZC(e⁻) + ZC(h⁺)
(4) ZC(e⁻) + O₂ → ·O₂⁻
(5) ·O₂⁻ + H₂O → ·OH + OH⁻
(6) ZC(h⁺) + OH⁻ → ·OH
(7) h⁺ + ·OH + ·O₂⁻ + TC → CO₂ + H₂O

CuO 的關鍵角色

  • 拓寬光吸收範圍（400→900 nm）
  • 加速光生 e⁻/h⁺ 分離（阻抗降 55%）
  • PN 接面內建電場驅動載子定向遷移

11 結論

研究結論

材料合成

共沉澱法成功合成 ZC 複合光觸媒。CuO 摻雜使能隙從 3.12 降至 2.78 eV，顯著改善光電性質。

光催化性能

ZC₀.₃ 在可見光下 60 min 內達 90.89% TC 去除率，為純 ZnO 的 4.99 倍。RSM 最佳化參數確定。

可見光響應

光吸收拓展至 900 nm。PN 異質結促進 e⁻/h⁺ 分離，增強催化活性。

活性物種

h⁺ 為主要活性物種，·O₂⁻ 和 ·OH 輔助。TC 礦化為無毒 CO₂ + H₂O。

實際應用

PMSS 固定化實現 10 次循環 >90%，20 天穩定 90%+，日光下效果與氙燈一致。

科學貢獻

為海水養殖廢水中 TC 光催化治理提供科學方向和技術支撐，具重要環境應用價值。

12 心得

個人心得與反思

值得學習之處

  • PN 異質結策略是解決寬能隙半導體可見光響應的經典而有效的方法
  • RSM 最佳化系統性地找到最佳參數組合，比逐一變量法更高效
  • PMSS 固定化巧妙解決粉體催化劑工程應用瓶頸 — 撈出即回收
  • 使用天然海水而非純水，大幅提升實際參考價值
  • 材料鑑定→降解途徑→毒性評估，研究鏈條非常完整

可以改進的地方

  • 毒性分析僅依賴軟體預測，缺乏實際生物毒性實驗
  • 未探討共存污染物對 TC 降解的影響
  • 日光驗證僅 1 小時，長期季節性影響需更多數據
  • 催化劑大規模製備可行性和成本分析尚待補充

對未來研究的啟發

  • 嘗試三元複合（如 ZnO/CuO/g-C₃N₄）提升可見光利用
  • 在台灣養殖場進行現場試驗，驗證真實水質效果
  • 探索連續流反應器設計，從批次走向工程應用
