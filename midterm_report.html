01 動機

養殖廢水中的抗生素危機

全球背景

  • 中國是全球最大抗生素生產與消費國，也擁有最大的海水養殖業
  • 四環素類（TCs）被大量用於水產養殖中的疾病防治與促進生長
  • 缺乏抗生素排放標準 → 養殖廢水直接排入環境

四環素（Tetracycline, TC）的危害

  • 促進抗藥性基因（ARGs）擴散
  • 通過食物鏈傳遞，對人類健康與永續發展構成重大風險
  • 傳統廢水處理技術難以有效去除，可能產生更毒副產物
  • 因此，解決海水養殖廢水中殘留抗生素刻不容緩

01 文獻

ZnO 光催化的挑戰與 CuO 摻雜策略

為什麼選 ZnO？

  • 光催化因高效且無二次污染，被認為最具前景
  • ZnO 具有成本低、光敏性高、穩定性好等優勢
  • 但 ZnO band gap 為 3.12 eV，僅能被紫外光激發，可見光利用受限

為什麼選 CuO 來摻雜？

  • CuO 是 p 型半導體（因氧空位缺陷），band gap 僅 1.2 eV
  • 對可見光有良好響應，成本低、化學穩定性優異
  • CuO（p 型）+ ZnO（n 型）→ 形成 PN 異質結
  • 內建電場促進光生電子-空穴對有效分離 → 更多自由基 → 更高效率

02 設計

奈米材料設計概念 — PN 異質結

設計思路

將 p 型半導體 CuO 與 n 型半導體 ZnO 結合，利用 PN 異質結的內建電場加速光生載子分離。

ZnO（n 型半導體）

  • 六方晶系結構，band gap 3.12 eV
  • ZnO 扮演電子供體角色，加速光生電子-空穴對分離

CuO（p 型半導體）

  • 單斜晶系結構，band gap 1.51 eV（文獻值 1.2 eV）
  • 因氧空位缺陷形成 p 型半導體特性

PN 異質結優勢

  • 界面內建電場驅動 e⁻ 和 h⁺ 向相反方向移動
  • 有效抑制光生電子-空穴對的複合
  • 更多載子參與光催化反應 → 更多自由基產生
  • band gap 降至 2.78 eV → 可見光全覆蓋（至 900 nm）

04 鑑定

XRD 結晶結構與 XPS 化學態

合成出來的材料到底是不是 ZnO/CuO 複合物？兩者之間有沒有真正的化學交互作用？
XRD 驗證晶體結構，XPS 檢測元素的化學狀態與電子轉移。

XRD — 確認「材料是什麼」

  • ZnO 特徵峰對應 JCPDS 01-070-2551（六方晶系）
  • CuO 特徵峰對應 JCPDS 01-080-1268（單斜晶系），隨摻雜比例增加而更明顯
  • 無雜峰 → 純度高，確認 ZnO + CuO 兩相成功共存

XPS — 確認「兩者有沒有交互作用」

  • Zn 2p、Cu 2p 峰 → 確認 Zn²⁺ 和 Cu²⁺ 存在
  • 關鍵發現：O 1s 結合能從 531.52 eV 正移 0.3 eV 至 531.82 eV
  • 正移 = ZnO 的電子雲密度降低 = 電子從 ZnO 轉移到 CuO
  • → 證實 ZnO 和 CuO 之間存在電子交互作用，PN 異質結確實形成

光催化降解結果

CuO 摻雜比例的影響

  • 暗反應 30 min：TC 幾乎不變 → 吸附可忽略
  • 純 ZnO 僅去除 18.21%，ZC₀.₃ 達 68.47%

08 途徑

TC 降解途徑與毒性評估

LC-MS/MS 中間產物鑑定

  • 使用三重四極桿液相層析串聯質譜儀分析
  • 鑑定出 9 個中間產物（P1–P9），建立完整降解路徑圖

主要降解過程

  • TC(P0) 的 C=C 雙鍵被自由基斷裂 → P1
  • P1 的羥基被 h⁺/自由基氧化 → P2、P3
  • 氨基碳鏈斷裂 → P4、P5
  • 六元碳環進一步斷裂 → P6、P7 → 被強氧化為 P8 → P9
  • 最終：P9 → CO₂ + H₂O

毒性評估與礦化率

  • ECOSAR 軟體（美國 EPA 驗證）模擬 96 h LC50（對魚）
  • 降解過程中可能產生比 TC 更毒的中間產物
  • 隨反應進行，中間產物進一步降解為低毒或無毒小分子
  • TOC 分析：反應 1 h 後由 5.95 降至 0.63 mg/L，礦化率 89.41%

09 機制

ZnO/CuO 光催化降解機制

PN 異質結光催化過程

  1. ZnO + CuO 形成 PN 異質結，調控能帶結構
  2. 可見光照射 → 電子從 VB 躍遷至 CB → 產生 h⁺
  3. CB 上的 e⁻ 與溶解氧反應 → 生成 ·O₂⁻
  4. ·O₂⁻ 與水反應 → 生成 ·OH
  5. VB 上的 h⁺ 氧化 OH⁻ → 額外生成 ·OH
  6. h⁺ + ·OH + ·O₂⁻ → TC 礦化為 CO₂ + H₂O

反應方程式

(3) ZC + hν → ZC(e⁻) + ZC(h⁺)
(4) ZC(e⁻) + O₂ → ·O₂⁻
(5) ·O₂⁻ + H₂O → ·OH + OH⁻
(6) ZC(h⁺) + OH⁻ → ·OH
(7) h⁺ + ·OH + ·O₂⁻ + TC → CO₂ + H₂O

CuO 的關鍵角色

  • 拓寬光吸收範圍（400→900 nm）
  • 加速光生 e⁻/h⁺ 分離（阻抗降 55%）
  • PN 接面內建電場驅動載子定向遷移
